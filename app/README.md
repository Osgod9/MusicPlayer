# 🎵 MusicPlayer

Aplicación Android desarrollada en Kotlin como proyecto de aprendizaje.

## Objetivo

Construir un reproductor de música desde cero, evolucionándolo paso a paso hasta convertirse en una aplicación con características similares a las de un reproductor moderno.

## Tecnologías

- Kotlin
- Android Studio
- XML
- ViewBinding
- MediaPlayer (próximamente Media3)
